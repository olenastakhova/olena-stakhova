**Цель сайта:** зашёл → почувствовал спокойствие и доверие → узнал квалификацию → записался.

## 3. Визуальная система (светлый минимализм)

Палитра выведена из фотосессии (беж/крем/терракотовая блуза/зелень растений) — сайт и фото
образуют единое целое.

| Токен | Цвет | Использование |
|---|---|---|
| `--paper` | `#FBFAF7` | основной фон |
| `--mist` | `#F3EFE9` | чередование секций, карточки |
| `--ink` | `#23211C` | заголовки, текст |
| `--stone` | `#6E6A61` | вторичный текст |
| `--clay` | `#BC5F3F` | ЕДИНСТВЕННЫЙ акцент: CTA, ссылки, детали |
| `--clay-deep` | `#9E4A2E` | hover |
| `--clay-tint` | `#F6E7DF` | подложки, теги |
| `--sage` | `#7C8B74` | микро-детали в векторной графике (листья) |
| `--line` | `#E5DFD6` | разделители, обводки |

**Типографика** (Google Fonts, поддержка latin-ext + cyrillic — обязательна для PL/UA/RU):
- Заголовки: **Playfair Display** (500/600) — тёплая элегантная антиква, «человечность».
- Текст/UI: **Inter** (400/500/600) — нейтральный, отличная читаемость.
- Шкала: 12 / 14 / 16 / 18 / 22 / 28 / 40 / 56–72 (clamp, fluid).

**Векторная графика (фирменный слой):** тонкий line-art, stroke 1.5px:
ветка/лист, концентрические дуги («спокойствие»), пунктирная траектория с точками («путь»),
мягкие органические блобы 3–4% насыщенности за фото. Никаких стоковых иллюстраций людей.

**Сетка:** контейнер 1140px, 12 колонок, щедрые отступы (секции 96–140px), скругления 20–28px,
тени почти невидимые (`0 12px 40px rgba(35,33,28,.07)`).

## 5. Анимации (сдержанные, «дышащие»)

| Где | Что | Как |
|---|---|---|
| Все секции | reveal: fade + translateY(24px), stagger 80ms | IntersectionObserver + CSS |
| Hero | line-art ветка «рисуется» (stroke-dashoffset), блоб медленно «дышит» (scale/rotate 12s) | SVG + CSS keyframes |
| Кнопки | лёгкий lift, стрелка сдвигается | CSS transition |
| Карточки | hover: подъём 4px, тень мягче, иконка подкрашивается | CSS |
| Счётчики | count-up при появлении | ~20 строк JS |
| Таймлайн | линия «прорастает» по скроллу | scroll-linked CSS/JS |
| Языки | смена текста с микро-fade | JS i18n |

Обязательно: `prefers-reduced-motion: reduce` — все анимации отключаются.
Никаких parallax-каруселей и тяжёлых библиотек: 60fps на мобильном.

## 6. Мультиязычность (PL default → UA, EN, RU)

- Польский — основной (SEO, локальный рынок). Коды: `pl` (корень `/`), `uk` (`/uk/`),
  `en` (`/en/`), `ru` (`/ru/`).
- В превью: клиентский переключатель (JSON-словари + `data-i18n`), выбор запоминается.
- В финальном сайте: **статические страницы на каждый язык** (лучше для SEO, работает без JS)
  + `hreflang` + локализованные `title/description`. Переключатель сохраняет позицию.

## 7. Технологический стек (реализация)

- **Astro 5** — современный статик-генератор: HTML без лишнего JS, встроенный i18n-роутинг,
  идеален для GitHub Pages. Альтернатива при желании «проще»: Vite + vanilla.
- Стили: **CSS custom properties** (+ опционально Tailwind 4 — решим на этапе реализации).
- Анимации: CSS + IntersectionObserver; при желании — **Motion One** (3KB).
- Форма: **Web3Forms** (бесплатно, без бэкенда, письма на email).
- Деплой: **GitHub Actions → GitHub Pages** (официальный Astro workflow), базовый путь
  `/<repo>/` или кастомный домен.
- SEO: schema.org `Psychologist/LocalBusiness`, OG-теги, sitemap, `hreflang`.
- Доступность: контраст AA, фокус-стили, семантика, alt-тексты.
